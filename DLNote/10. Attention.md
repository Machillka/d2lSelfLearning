# Q, K V

自主性提示： Query
非自主性提示：Key
感官输入：Value
### 整体联系

查询 - **注意力汇聚** - 感官输入

通过 Query 与 Key 的匹配，寻找到最佳匹配值 Value

每个 Key 对应唯一 Value


# 一般化模型

$$
f(q) = \sum_i\alpha(q, k_i)v_i
$$
其中 $q$ 为 查询, $（k_i, v_i）$ 为一个键值对

# 理解

根据公式，注意力机制相当于根据当前的查询，对V进行加权求和，其中的权重就是 $\alpha (q, k_i)$ 函数。 表示自主性提示和非自主性提示的汇聚 (Pooling)
